## Definition
Open-Set Recognition (OSR) is the ability of a model to correctly identify when an input sample belongs to a class it was *not* trained on, distinguishing it from known classes. This is vital for security systems.

## How It Applies Here
This technique was utilized by Nguyen Thanh Minh Duy to achieve high detection accuracy for unknown/unseen classes in open-set environments, crucial for detecting zero-day network attacks.
